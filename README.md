Inkle Multi-Agent Tourism Planner

This is a multi-agent AI project that helps users plan trips by fetching, in real time, the weather and nearby tourist attractions for any city in the world.

Features
Parent Tourism Agent
Understands user queries and orchestrates child agents.
Weather Agent
Gets live temperature and rain probability using Open-Meteo API. Places Agent
Suggests up to 5 tourist attractions using Overpass API (OpenStreetMap).
Geocoding Agent
Converts place names to coordinates using the Nominatim API.
Error Handling
If it doesn't exist, the system responds with:
"I don't know whether this place exists."
Example Queries
“I am going to Bangalore, what about the temperature?”
"Plan my trip to Bangalore. “What are the places I can visit and what’s the weather?” APIs Used Nominatim (Geocoding) Open-Meteo (Weather) Overpass API (Tourist Places)
